Introduction

SandboxCodex is an AI-powered platform for automated code review, testing, and PR generation.

Installation

Get started by installing SandboxCodex:

npm install sandboxcodex

Quick Start

Create your first project in 3 steps:

sandboxcodex init my-project cd my-project npm start

Configuration

Configure SandboxCodex via sandbox.config.js:

module.exports = { projectName: 'my-project', sandbox: { runtime: 'nodejs' }, github: { token: process.env.GITHUB_TOKEN } };
